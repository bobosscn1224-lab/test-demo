# Quality Checklist

Before final delivery, check the following.

## Visual fidelity

- [ ] Canvas ratio matches source.
- [ ] Main margins match source.
- [ ] Title position and hierarchy match source.
- [ ] Background color/gradient is close.
- [ ] Card positions and sizes are aligned.
- [ ] Image crops are not stretched.
- [ ] Logos and screenshots are faithful.
- [ ] No visible random artifacts.

## Text correctness

- [ ] Chinese characters are correct.
- [ ] Numbers, units, dates, and names are correct.
- [ ] No OCR garbage.
- [ ] Line breaks are intentional.
- [ ] Font size is readable.

## Editability

- [ ] Main headings editable.
- [ ] Body text editable when legible.
- [ ] Cards/lines editable.
- [ ] Simple charts/tables editable when feasible.
- [ ] Complex assets are documented as image-based.

## Output files

- [ ] `reconstructed.pptx` opens.
- [ ] `reconstruction.svg` opens.
- [ ] `layout_plan.json` validates.
- [ ] `assets/` contains required crops.
- [ ] `qa_report.md` states limitations.
