## 行业标准 5+1 闭环流程

```
Step0 剧本结构化 → Step1 资源资产固化 → Step2 标准化分镜拆解
→ Step3 全局导演风格锁定 → Step4 逐镜头可控生成 → Step5 全自动成片合成
```

## 现有框架 vs 行业标准 — 差距分析

| 步骤 | 行业标准 | 我们的现状 | 差距 |
|------|---------|-----------|------|
| Step0 | 故事→可拍摄剧本（去文学化、拆人物小传） | 直接粘贴原文，AI 尽力解析 | **缺失** |
| Step1 | 全局资产库：定妆/场景/道具 永久ID+标签+版本管理 | 单项目内生成，无跨项目复用 | **薄弱** |
| Step2 | 资源自动绑定分镜，可复用模板 | ✅ 已实现 | — |
| Step3 | 全片风格一键锁定 | ✅ 一致性圣经 | — |
| Step4 | 批量可控生成 + 智能容错 | 逐个点击生成，单一重试 | 可优化 |
| Step5 | AI剪辑+配乐+字幕+调色→直接出片 | **完全缺失** | **最大短板** |

## 改进方案（按优先级排列）

### P0 — 补齐闭环：AI 自动成片模块

- 按分镜顺序拼接所有镜头
- 自动匹配转场 + 背景音乐 + 环境音效
- 自动生成字幕 + 字幕动画
- 全局统一调色、降噪、补帧
- 一键导出 1080P/4K 成片

### P0 — 剧本结构化预处理（Step0）

- AI 将小说/故事自动改写为标准影视短剧剧本
- 自动拆分：单人动作、双人互动、空镜、情绪镜头、转场镜头
- 自动剔除无法影视化的心理描写、旁白文字
- 适配 3 分钟短剧节奏，控制总镜头数和单镜头时长

### P1 — 资源管理体系重构

- 全局资产唯一 ID + 风格/光影/年代标签
- 跨项目复用：同角色多版定妆，一键切换
- 智能资源补全：AI 自动检测缺失资源并批量生成
- 资源合规校验：自动检测人脸合规、画面违规

### P1 — 小白体验优化

- 模板化开局：甜恋/治愈/悬疑/校园/都市等短剧模板
- 新手引导：每步悬浮提示
- 智能容错：自动分析失败原因、修复 Prompt、重试
- 一键全自动模式：粘贴故事 → 全流程自动 → 输出成片

### P2 — 一致性机制强化

- 锁定人物：五官/发型/体型/服饰/肤色/表情基准
- 锁定画面：分辨率/画质/胶片感/锐度/饱和度
- 锁定光影：柔光/硬光/黄昏/室内光统一不变
- 全局一键继承：第一个镜头的风格自动同步所有镜头

---

## 最终目标：小白只需 3 步

1. **粘贴故事** → AI 自动转标准剧本、拆解角色场景道具
2. **一键生成资源** → 锁定人物/场景/道具风格
3. **一键成片** → AI 分镜 + 导演 + 逐镜生成 + 剪辑配乐 + 输出成品
